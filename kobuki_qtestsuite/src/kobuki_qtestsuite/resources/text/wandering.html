Overview

Puts the kobuki into a wandering mode - useful for checking sound levels as well as cliff/bump detection capabilities.

Bump Test

Put it aground and start wandering. The bump sensor will reorient the robot in a random direction every time it is triggered.

Cliff Test

This is easier to test by picking up the robot, however cliff and wheel drop sensors will automatically disable the robot if the commanded speed is high, so make sure you set this to a low value.
  • Set linear speed to 0.1
  • Pick up the robot and flip it over
  • Click the start button
  • Wave your hand over the cliff sensors
